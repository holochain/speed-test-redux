## Overview

A dna to duplicate the speed-test used in RSM for benchmark comparison purposes.

Assumes you are running inside `nix-shell` executed from [holochain-redux](https://github.com/holochain/holochain-rust)

``` shell
cd test
npm i
npm run test:stress
```
